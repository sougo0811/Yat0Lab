## 環境構築
### 仮想環境の作成
```
# Windows
python -m venv 仮想環境名
# Linux and OS X
python3 -m venv 仮想環境名
```
### 仮想環境の起動
```
# Windows
仮想環境名\Scripts\activate
# Linux and OS X
source 仮想環境名/bin/activate
#自分用
source yatoivenv/bin/activate
```
### pipアップグレード
```
python -m pip install --upgrade pip
```
